# H5 中间帧编辑器

线上地址：`https://<部署域名>/h5/`（PC / 手机同地址，自适应）

## 给使用者

- **入口**：飞书多维表格每行「调整链接」列直达该记录编辑器；或打开 `/h5/` 首页选记录；或在首页直接上传视频建任务（自动分片，≤500MB）
- **画布**：按真实视频分辨率等比显示最后一帧；点选图层 → 拖拽移动、角点缩放
- **图层**：搜索框（来自表格「搜索框图片」列）、引导语（文案池下拉/自定义/留空随机）、角色名（自动读「角色名」列，列空则成片隐藏）、自定义字幕、图片（来自共享素材库或新上传）
- **文字样式**：字号、字色/描边/阴影（色板 + 系统取色器 + Hex/RGB）；阴影含不透明度与模糊度
- **图片动画**：抖动 / 上下浮动 / 闪烁，画布实时预放，成片由 ffmpeg 渲染
- **显示时机**：每个图层可选「全程显示」或「跟随引导语第 N 段」
- **按钮**：精确预览（服务端与成片同渲染器出图）→ 保存样式（写回「样式参数」列）→ 设为默认（同规格素材全员生效）→ 提交处理（处理这一条并轮询出片）
- **主题**：右上角深色/浅色切换，本地记忆

## 给维护者

- 前端：`static/h5/index.html` 单文件（Konva CDN），由 `main.py` 挂载 `/h5/`
- 后端：`src/api/h5.py`（路由 `/api/v1/h5/*`），渲染核心 `src/tools/layer_render.py`，数据模型 `src/tools/layer_model.py`
- 样式参数：记录级（表格「样式参数」列 JSON）> 全局默认（PostgreSQL h5_kv）> 内置默认（layer_model.BUILTIN_DEFAULT_LAYERS，与图层化改造前的固定样式一致）
- 帧缓存：h5_kv 表 `frame:<md5>`；素材库：h5_assets 表
- 上传：扣子网关请求体约 10MB 上限，大文件走 init/chunk/finish 三段式（6MB/片）
- 图片加载不要设 crossOrigin（TOS 预签名无 CORS 头）；模型网关恒返回 SSE 流（见 `_vision_detect_subtitle`）
